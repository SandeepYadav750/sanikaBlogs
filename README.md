# 📝 Sanika Blogs - A Modern Blogging Platform

A full-stack blogging platform built with Next.js 14, Node.js, MongoDB, and Tailwind CSS. Users can create, read, update, and delete blogs, write comments, and manage their profiles.

## ✨ Features

- 🔐 **Authentication** - JWT-based authentication with secure cookies
- 📝 **Blog Management** - Create, read, update, delete blog posts
- 💬 **Comments System** - Engage with posts through comments
- 👤 **User Profiles** - Customize profile with avatar and bio
- 🌓 **Dark/Light Mode** - Theme switching for better user experience
- 📱 **Responsive Design** - Fully responsive on all devices
- 🚀 **Performance** - Optimized with Next.js App Router

## 🛠️ Tech Stack

### Frontend
- Next.js 14 (App Router)
- Redux Toolkit for state management
- Tailwind CSS for styling
- Axios for API calls
- React Toastify for notifications

### Backend
- Node.js with Express
- MongoDB with Mongoose
- JWT for authentication
- Bcrypt for password hashing

### Deployment
- **Frontend:** Vercel
- **Backend:** Render
- **Database:** MongoDB Atlas

## 📋 Prerequisites

Before you begin, ensure you have:
- Node.js (v18 or higher)
- npm or yarn or pnpm
- MongoDB Atlas account (or local MongoDB)
- Git
